# telnet的应用
## TCP长连接
```c
telnet IP PORT
```
如上所示，使用上诉方法，可以打一条TCP的长连接。
## 登录设备
# telnet 的退出
telnet 的退出 分成两种，一中是在telnet命令中，直接输入 quit 或者 q 即可退出。 二种情况是 已经进入了端口中，需要先从端口中退出，然后再退出telnet。
## telnet 直接退出
输入 q 或者 quit 即可退出。

![](attachments/Pasted%20image%2020230508123849.png)

## telnet 退出端口后退出telnet
1.  输出 ctrl + ]
2.  q 或者 quit
